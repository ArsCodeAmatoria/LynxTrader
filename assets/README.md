# Assets Directory

This directory contains visual assets for the LynxTrader project documentation.

## Required Images

### 1. Banner Image (`lynxtrader-banner.png`)
- **Dimensions**: 800x400px (2:1 aspect ratio)
- **Style**: Cyber/futuristic theme with neon colors
- **Content**: LynxTrader logo with cyberpunk aesthetic
- **Colors**: Primary: Cyan (#00FFFF), Green (#39FF14), Pink (#FF007F)
- **Background**: Dark cyber gradient (#0F0F1A to #1A1A2E)

### 2. Dashboard Preview (`dashboard-preview.png`)
- **Dimensions**: 900x600px (3:2 aspect ratio)
- **Style**: Screenshot of the cyber dashboard interface
- **Content**: Main trading dashboard with:
  - Mathematical formulas (KaTeX rendered)
  - Glowing cards and neon UI elements
  - Charts with cyber styling
  - AI risk management section
  - Matrix rain effects

## Design Guidelines

### Color Palette
```
Cyber Blue:   #00FFFF
Matrix Green: #39FF14
Neon Pink:    #FF007F
Neon Orange:  #FF6600
Neon Purple:  #BF00FF
Neon Yellow:  #FFFF00

Background:   #0F0F1A
Secondary:    #1A1A2E
Accent:       #16213E
```

### Typography
- **Headers**: Orbitron (futuristic sans-serif)
- **Code/Math**: Fira Code (ligature-enabled monospace)
- **Matrix**: Share Tech Mono (cyberpunk monospace)

### Effects
- Glow effects with multiple shadow layers
- Semi-transparent backgrounds with backdrop blur
- Gradient borders and text
- Animated elements (scan lines, matrix rain)

## Image Creation Tools

Recommended tools for creating these assets:
- **Figma**: For UI mockups and screenshots
- **Adobe Photoshop**: For complex graphics and effects
- **GIMP**: Free alternative for image editing
- **Canva**: For quick banner creation
- **Screenshot Tools**: For capturing actual dashboard

## Notes

- All images should maintain the cyber/futuristic aesthetic
- Use high-resolution images for crisp display
- Optimize file sizes for web without losing quality
- Ensure images work well on both light and dark backgrounds
